# Borradores de investigación: no enviar

Estos archivos no acreditan consentimiento ni una relación contractual previa.
No deben enviarse como publicidad no solicitada. Se conservan únicamente como
material de investigación y solo pueden adaptarse si la clínica pide una
auditoría, autoriza el contacto o una revisión jurídica documenta otra base
aplicable.

Antes de responder a una solicitud real, registrar en el CRM:

- Fuente y fecha.
- Texto o prueba del consentimiento.
- Canal autorizado.
- Responsable de la respuesta.
- Oposición o baja, si se produce.
